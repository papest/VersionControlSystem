Reverse engineering

Write a function called reverseWords that takes a sentence as input and reverses the order of the words using StringBuffer. The function should return the reversed sentence.

If you are up for a challenge, try solving this without using any other built-in string manipulation or reverse methods.


Sample Input:
I love kotlin!

Sample Output:
kotlin! love I


Sample Input:
Welcome to StringBuffer universe

Sample Output:
universe StringBuffer to Welcome



Memory limit: 256 MB
Time limit: 8 seconds
